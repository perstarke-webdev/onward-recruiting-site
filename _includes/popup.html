Diese Website wird über GitHub Pages zur Verfügung gestellt, was bedeutet, dass sie ein globales Verteilungsnetzwerk (CDN) nutzt, um Inhalte zuverlässig anzuzeigen. Wenn Sie die Website nutzen, können einige Ihrer nicht-personenbezogenen Daten, wie z. B. die IP-Adresse und Informationen zur Seitennutzung, über dieses Netzwerk übertragen werden, um das ordnungsgemäße Funktionieren der Website zu gewährleisten. Mehr Informationen dazu finden Sie im Impressum

Bitte bestätigen Sie, dass Sie damit einverstanden sind und dass wir ein Zustimmungs-Cookie verwenden, das für die Dauer von einem Tag gespeichert wird. Dieses Cookie ermöglicht es uns, Ihre Zustimmung zur Verwendung nicht-personenbezogener Daten zu speichern und das ordnungsgemäße Funktionieren der Website zu gewährleisten
