Information zur Nutzung von GitHub Pages

Diese Website wird über GitHub Pages bereitgestellt und nutzt ein globales Verteilungsnetzwerk (CDN). Bei der Nutzung der Website werden einige Ihrer nicht-personenbezogenen Daten, wie z.B. die IP-Adresse und Informationen zur Seitennutzung, über dieses Netzwerk übertragen, um das ordnungsgemäße Funktionieren zu gewährleisten. Mehr Informationen finden Sie im Impressum.

Essentielles Cookie

Wir verwenden ein essentielles Cookie, um Ihre Präferenzen bezüglich der Datenverarbeitung auf unserer Website zu speichern. Dieses Cookie ist für die Funktionalität unseres Cookie-Banners notwendig und wird für 24 Stunden gespeichert. Es enthält keine personenbezogenen Daten. Bitte bestätigen Sie, dass Sie diese Information zur Kenntnis genommen haben. Diese Bestätigung bezieht sich nur auf das essentielle Cookie und stellt keine Zustimmung zur Verwendung weiterer Cookies oder zur Verarbeitung personenbezogener Daten dar. Weitere Cookies werden auf unserer Website nicht verwendet.
